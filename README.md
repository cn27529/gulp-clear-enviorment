# gulp-clear-enviorment v1.0

---

使用 webpack v3.3版本,

外部導入

```
// ver15.1.0
react.js / react-with-addons.js / react-dom.js
```

## 安裝方式

- 初次使用方式：

    * 安裝全域 **gulp**及**node-sass**

        ```
        $> npm install -g gulp


        $> npm install -g node-sass

        ```

- 安裝套件方式：

    ```
    $> npm install
    ```

---

## 執行gulp開發任務：

```
//執行local server 開發環境
$> gulp server

//執行build動作，產生dist包
$> gulp build
```
